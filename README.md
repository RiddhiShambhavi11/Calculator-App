Calculator App

Overview

A modern Android Calculator application built using Kotlin and Jetpack Compose. The app performs basic arithmetic operations with a clean, responsive, and user-friendly interface.

Features

- Addition, subtraction, multiplication, and division
- Modern UI built with Jetpack Compose
- Responsive layout
- Simple and intuitive user experience

Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio
- Material Design 3
